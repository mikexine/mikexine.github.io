About


Hello, nice to meet you!

I am Michele Colombo, born in Italy some years ago. Currently I live in Copenhagen - Denmark, where I am attending the Master of Science in Business Administration and Information Systems - E-Business at Copenhagen Business School. I moved to Copenhagen after I finished my Bachelor degree in Economics and Management at the University of Padua, in Italy.

Through this website, you can have a look at some small projects that I am developing. I hope you'll like that!

I just started to occasionally publish some posts on medium. My account is @_m.


Telegram Bots

  • @TheRockBot: you can ask it some data about The Rock Trading Ltd. If you want, you can send your The Rock API keys to the bot: you'll be able to check your balances, trades and transactions. This bot is not official.g

  • @UnipdBot: an unofficial bot that knows lots of useful data about the University of Padua: study rooms, libraries, and more! You can find its reviews on the StoreBot website. The source code is on GitHub.


Python

  • unidata: providing data about Italian universities in a simple, accessible way. The source code is on GitHub.

  • TheRock-Py: unofficial Python wrapper for The Rock Trading APIs. It currently supports both Python 2.x and 3.x, and all the API methods on The Rock.

Ionic

  • Unipd App: an app that provides the same data provided by @UnipdBot. Available in Alpha mode for android. iOS version is not available yet - the Apple's license is expensive! The source code is on GitHub.


Instagram Maps

  • Europe In Erasmus: a project launched in August 2014 and managed with a friend (@giuseppecik). You can view all the pictures on Instagram with the #inErasmus hashtag. At the moment it is discontinued; I'm planning to rewrite the underlining code and republish it.
