About


Hello, nice to meet you!

I am Alessandro Michele Colombo, born in Italy some years ago. Currently I live in Padua where I am attending the Master of Science in Economics and Finance and where I gained also my Bachelor degree in Economics and Management.

Through this website, you can have a look at some small projects that I am developing. I started to learn Python, HTML, CSS, Git and other "computer things" on my own, since I really wanted to be able do something more with my laptop.


Telegram Bots

  • @TheRockBot: you can ask it some data about The Rock Trading Ltd..

  • @TheRockBot: this bot allows you to trade any currency officially supported on ShapeShift.io to any other ShapeShift currency!


Python things

  • TheRock-Py: unofficial Python wrapper for The Rock Trading APIs. It currently supports bot Python 2.x and 3.x, and all the API methods on The Rock. Complete support for Websockets is under development and will be ready soon.

  • Trt Last: a small script that generates an image with the price of the last trade on the BTC/EUR market on The Rock Trading.

  • secret.py: stay tuned! Great things will follow soon...


InstaMaps

  • Europe In Erasmus: a project launched in August 2014 and developed managed with a friend (@giuseppecik). You can view all the pictures on Instagram with the #inErasmus hashtag.

  • UniPd Map: a map with the #unipd hashtag! This hashtag stands for "University of Padua.

  • ..your map? If you have a cool idea on what should be mapped, contact me! I'll do it for the fun of doing it, if I like the idea!


The code for most of the things linked here is available through my GitHub account. If something is missing, it's probably because I forgot to add it or because I am rewriting it; I don't like the idea of hiding my code in my basement, so if you might need something and I haven't published it yet, drop me an email and I'll reply soon.
